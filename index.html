How to Install

Because GastroBot is not on the Google Play Store, you'll need to enable one setting to install it. It's simple and safe!

1Download the File

Tap the download button above. Your browser will save the `GastroBot.apk` file to your phone.

2Allow Unknown Apps

Open the downloaded file. Your phone will prompt you for permission to install from your browser. Tap "Settings" and enable it.

3Install & Enjoy!

Go back, tap "Install," and you're all set! Open GastroBot and start creating amazing recipes.
